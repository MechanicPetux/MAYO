# MAYO
Всё-в-одном коллекция минификаторов

## Использование
Приложение доступно [здесь](https://mechanicpetux.lol/mayo/)<br>
Вы можете выбрать файлы, или перетащить их в любое место на странице.

Немного подождите... И всё что вы выложили будет скачано обратно в сжатом виде. Проще не бывает

Все возникшие ошибки будут вам показаны.

## Поддерживаемые языки
Всё, чему больше всего нужна минификация: JSON, JavaScript, CSS, HTML.

## Использовано
[Terser](https://github.com/terser/terser) - JS минификатор<br>
[CSSO](https://github.com/css/csso) - CSS минификатор<br>
[HTMLMinifier](https://github.com/terser/html-minifier-terser) - HTML минификатор<br>
[JSZip](https://github.com/Stuk/jszip) - Поддержка Zip<br>
