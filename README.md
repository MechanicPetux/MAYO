# MAYO
All-in-one minifier collection

[![Русский Перевод](https://mechanicpetux.lol/stuff/Shield_ru.svg)](ReadMe.ru.md)

## Usage
The app is avaliable [here](https://mechanicpetux.lol/mayo/)<br>
You can select files, or drag and drop them on any place on the page.

Wait a little bit... And all you uploaded will be downloaded back minified. It couldn't be more simple

Any errors occurred will be displayed to you.

## Supported languages
All that that needs minification most: JSON, JavaScript, CSS, HTML.
